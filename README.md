Проект можно собрать с помощью Apache Ant 1.10.12 (Файл build.xml)

Команда для сборки: ant


Также проект можно собрать с помощью Apache Maven 3.9.12 (Файл pom.xml)

Команда для сборки: maven package


Для сборки проекта подойдёт Java 8+.


Для запуска проекта можно использовать команду:

java -jar util.jar -s -f -a -p sample- in1.txt in2.txt


Или можно использовать один из приложенных скриптов shift.bat для Windows или shift.sh для Linux.
